### Git 常用更新指令手冊

**【情況一：日常更新程式碼】**
當你修改了檔案，想要同步到 GitHub 時，請依序執行：

1. **查看目前的修改狀態** (確認改了哪些檔案)
   `git status`

2. **將所有變更加入暫存區**
   `git add .`

3. **提交並紀錄更新說明** (引號內可自行修改)
   `git commit -m "更新說明：修正了某個功能"`

4. **推送到 GitHub**
   `git push`

---

**【情況二：常用的輔助指令】**

* **如果想撤銷還沒 add 的修改：**
  `git checkout -- 檔名`

* **如果想看之前的提交紀錄：**
  `git log --oneline`

* **如果 remote 設定錯誤，要重新連結網址：**
  `git remote set-url origin 你的GitHub網址`

* **從 GitHub 抓取最新的檔案下來：**
  `git pull`
